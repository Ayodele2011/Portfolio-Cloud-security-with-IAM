# Portfolio: Cloud Security with IAM

This repository showcases practical projects on **AWS Cloud Security** using **Identity and Access Management (IAM)**, **Amazon S3**, and **Virtual Private Cloud (VPC)**. It is part of my portfolio demonstrating hands-on skills in securing cloud environments.

---

## 📂 Projects Included
- **Access S3 from a VPC**  
  Demonstrates how to securely connect to Amazon S3 from within a Virtual Private Cloud.

- **Cloud Security with AWS IAM**  
  Explains IAM roles, policies, and best practices for managing access securely.

- **Host a Website on S3**  
  Walkthrough of hosting a static website on Amazon S3 with proper security configurations.

---

## 🚀 Skills Demonstrated
- AWS Identity and Access Management (IAM)
- Secure access control policies
- Amazon S3 bucket configuration
- Hosting and securing static websites
- Cloud networking with VPC

---

## 📖 How to Use
1. Browse the PDF files in this repository for step-by-step guides.
2. Follow along in your own AWS account to replicate the configurations.
3. Use the examples as templates for securing your own cloud projects.

---

## 🔒 Security Best Practices
- Principle of least privilege for IAM roles
- Enforcing HTTPS for S3-hosted websites
- Restricting access to S3 buckets from specific VPCs
- Regularly reviewing IAM policies
